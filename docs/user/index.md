# 用户使用手册

## 1. 快速开始

### 1.1 界面介绍

小当家的主界面分为以下几个区域:

```
+----------------------------------------------------------+
+  标题栏 (Title Bar)                                     ++
+----------------------------------------------------------+
+  菜单栏 (Menu Bar)                                      ++
+----------+-----------------------------------------------+
+          ++                                               ++
+  侧边栏  ++            画布区域 (Canvas)                 ++
+          ++                                               ++
+ (Sidebar)++           脑图式文件节点展示                 ++
+          ++                                               ++
+          ++                                               ++
+----------+-----------------------------------------------+
+                   状态栏 (Status Bar)                    ++
+----------------------------------------------------------+
```

### 1.2 创建第一个项目

1. 点击左侧 **"新建项目"** 按钮
2. 输入项目名称
3. 选择要管理的文件夹目录
4. 点击 **"创建"**

系统将自动扫描目录并在画布上显示文件节点。

## 2. 基本操作

### 2.1 项目管理

| 操作 | 说明 |
|------|------|
| 新建项目 | 创建新的文件管理项目 |
| 切换项目 | 在侧边栏点击项目名称 |
| 删除项目 | 右键项目 → 删除 |
| 刷新项目 | 右键项目 → 重新扫描 |

### 2.2 文件浏览

| 操作 | 说明 |
|------|------|
| 单击选中 | 选中文件节点 |
| 双击打开 | 打开文件阅读/编辑器 |
| 右键菜单 | 查看更多操作选项 |

### 2.3 画布操作

| 操作 | 说明 |
|------|------|
| 鼠标滚轮 | 缩放画布 |
| 空格 + 拖动 | 平移画布 |
| 拖动节点 | 移动文件节点位置 |
| 框选 | 鼠标框选多个节点 |

## 3. 画布交互

### 3.1 节点拖拽

1. 鼠标按住文件节点
2. 拖动到目标位置
3. 松开鼠标

节点位置会自动保存，下次打开项目时保持不变。

### 3.2 缩放与导航

- **放大**: 鼠标滚轮向上滚动 或点击 `+` 按钮
- **缩小**: 鼠标滚轮向下滚动 或点击 `-` 按钮
- **适应画布**: 点击 `适应窗口` 按钮
- **实际大小**: 点击 `100%` 按钮回到原始大小

### 3.3 节点详情

鼠标悬停在节点上可查看:
- 文件名
- 文件大小
- 文件类型标签
- 关联文件数量

## 4. 文件操作

### 4.1 打开文件

双击文件节点，将在底部面板打开文件阅读器:

- **代码文件**: 使用 Monaco Editor 高亮显示
- **图片文件**: 显示图片预览
- **文本文件**: 显示纯文本内容
- **其他文件**: 显示文件信息，可选择用系统默认程序打开

### 4.2 编辑文件

1. 双击打开文件
2. 在编辑器中修改内容
3. 按 `Ctrl + S` 保存

### 4.3 重命名与删除

右键文件节点:
- **重命名**: 修改文件名
- **删除**: 从文件系统中删除文件
- **在资源管理器中显示**: 打开系统文件管理器

## 5. AI 智能功能

### 5.1 标签系统

系统会自动为文件生成标签:

| 标签类型 | 示例 |
|----------|------|
| 编程语言 | JavaScript, Python, Rust |
| 文件类型 | 图片, 视频, 文档, 配置 |
| 角色 | 测试文件, 入口文件 |
| 状态 | 大文件, 隐藏文件 |

### 5.2 文件关联分析

右键项目名称 → **"分析关联"**

系统将分析文件之间的关联关系:
- 导入/依赖关系
- 引用关系
- 命名相似
- 同目录/同类型

### 5.3 查找相似文件

1. 选中文件节点
2. 右键 → **"查找相似"**

系统将列出与当前文件相似的文件列表。

### 5.4 智能搜索

在顶部搜索框中输入关键词:

- 支持文件名搜索
- 支持标签搜索 (如 "Python")
- 支持扩展名搜索 (如 ".json")
- 支持组合搜索 (如 "Python test")

## 8. 常见问题

### 8.1 文件不显示

**解决方法**:
1. 确认选择的目录路径正确
2. 右键项目 → 重新扫描
3. 检查目录是否有访问权限

### 8.2 节点位置丢失

**解决方法**:
1. 系统会自动保存节点位置
2. 如遇异常，可重新扫描目录恢复

### 8.3 大型目录加载慢

**优化建议**:
- 对于包含上万文件的目录，首次扫描可能需要一些时间
- 可在扫描前手动整理目录结构

### 8.4 应用无法启动

**检查项**:
1. 确认系统已安装 WebView2
2. 以管理员权限运行一次
3. 检查杀毒软件是否拦截
